# PPT Showcase Library

Use this folder for reusable visual examples that teach deck grammar.

Each showcase should include:

- a preview image or HTML example
- deck type: consulting, pitch, keynote, technical report, casebook, training, or image-driven
- page archetype: cover, thesis, divider, comparison, architecture, case story, case proof, scorecard, roadmap, or closing
- visual grammar notes: typography, grid, palette, image treatment, footer logic
- reuse warning: what should not be copied blindly

Recommended starter set:

| Example | Purpose |
| --- | --- |
| executive-cover | premium first impression with restrained hierarchy |
| dense-proof | high-information page that stays readable |
| case-spread | narrative page plus evidence page |
| architecture-map | technical system without tiny-label clutter |
| launch-moment | product/keynote page with strong visual memory |
| ink-tech-report | Chinese technology/report visual language |

Do not add generic decorative pages. A showcase is useful only if future decks can learn a repeatable move from it.
