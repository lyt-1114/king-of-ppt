# king-of-ppt

This repository is organized around two skills.

Start here:

- [skills/ultimate-ppt/SKILL.md](skills/ultimate-ppt/SKILL.md)
- [skills/image2ppt-exact/SKILL.md](skills/image2ppt-exact/SKILL.md)

## What Lives Here

### 1. `ultimate-ppt`

Primary skill for strategy-first deck creation, deck rebuilding, image-driven presentation work, keynote decks, sales decks, reports, and HTML/PPTX delivery paths.

Entry:

- [skills/ultimate-ppt/SKILL.md](skills/ultimate-ppt/SKILL.md)

Supporting files:

- `skills/ultimate-ppt/references/`
- `skills/ultimate-ppt/assets/`
- `skills/ultimate-ppt/scripts/audit_deck.py`

### 2. `image2ppt-exact`

Focused skill for exact image-deck reproduction, OCR-based editable text recovery, and blueprint-based high-fidelity PPT rebuilds.

Entry:

- [skills/image2ppt-exact/SKILL.md](skills/image2ppt-exact/SKILL.md)

Implementation package:

- [packages/image2ppt-exact/README.md](packages/image2ppt-exact/README.md)
- `packages/image2ppt-exact/src/image2ppt_exact/`

## Install

```bash
npx skills add https://github.com/lyt-1114/king-of-ppt
```

## Recommended Navigation

If you want the main presentation workflow, open `ultimate-ppt`.

If you want exact image-to-PPT reconstruction, open `image2ppt-exact`.

If you want Python implementation details, tests, or CLI code, go to `packages/image2ppt-exact`.
